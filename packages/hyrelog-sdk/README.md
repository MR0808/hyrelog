# @hyrelog/sdk

Official TypeScript/JavaScript SDK for the **HyreLog Audit Trail API**.

HyreLog is a production-grade audit log backend for SaaS platforms and internal tools.  
This SDK wraps the HyreLog HTTP API with a typed client.

---

## Installation

```bash
npm install @hyrelog/sdk
# or
yarn add @hyrelog/sdk
# or
pnpm add @hyrelog/sdk
```
